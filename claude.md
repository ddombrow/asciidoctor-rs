# Project: asciidoctor-rs

The goal is to build a rust port of asciidoctor and asciidoctor-js in rust.  I have cloned the original asciidoctor repo at ~/src/asciidoctor

## Other Goals
- Follow the language spec, I have cloned that repo at ~/src/asciidoc-lang
- Pass the asciidoc TCK, I have cloned that repo at ~/src/asciidoc-tck
- Build a react component that uses a wasm built from our Rust code, it should be compatible with the existing react-asciidoc, whose repo I have cloned at ~/src/react-asciidoc
- Be able to render all adoc examples in the examples folder and subdirectories
