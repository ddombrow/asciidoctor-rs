= Fenced Code Blocks Demo

This file exercises Markdown-style fenced code blocks.

== Plain fenced block

```
puts "Hello from a plain fence"
```

== Fenced block with language

```rust
fn main() {
    println!("Hello from a Rust fence");
}
```

== Fenced block with language and option

```javascript,linenums
console.log("Hello from a line-numbered fence");
```
