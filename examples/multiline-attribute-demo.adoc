= Multiline Attribute Demo
:description: If you have a very long line of text \
that you need to substitute regularly in a document, \
  you may find it easier to split the value neatly in the header \
  so it remains readable in the source.
:haiku: Write your docs in text, + \
  AsciiDoc makes it easy, + \
  Now get back to work!

This example is meant to be inspected using the JSON-oriented outputs.

The multiline header attributes above exercise both soft wraps and hard wraps.

:body-note: Body attributes can also be continued \
  before later blocks are parsed.

Run with `cargo run -- --format json examples\multiline-attribute-demo.adoc`
to inspect the collected document attributes.
