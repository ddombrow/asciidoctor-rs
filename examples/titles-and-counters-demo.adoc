This example combines stacked block attribute lists, captioned titles, custom
caption overrides, and plain block title counters in one document.

== Captioned titles

.Example blocks are captioned by default
====
Example titles get the built-in Example counter.
====

[source,rust]
----

    a + b
}

== Custom caption overrides

.Custom caption starts an alphabetic counter
[caption="Example {counter:demo-example:A}: "]
====

====

.The same custom caption counter increments on the next block
[caption="Example {counter:demo-example}: "]
====
The named counter continues from A to B.
====
