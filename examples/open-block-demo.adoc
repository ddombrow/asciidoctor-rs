== Bare Open Block

A bare open block groups content without adding any visual styling.

--
This paragraph is inside an open block.

So is this one.

And this listing block:

----
puts "hello from inside an open block"
----
--

Back to normal content.

== List Continuation

The main practical use of open blocks is attaching multiple blocks to a list item.

. Clone the repository:
+
--
----
git clone https://github.com/example/myproject
cd myproject
----

NOTE: Use SSH if you have two-factor auth enabled.
--

. Build the project:
+
--
----
cargo build --release
----

The binary will be at `target/release/myproject`.
--

. Run the tests:
+
----
cargo test
----
