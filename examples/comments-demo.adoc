== Comments Do Not Strip Content Inside Verbatim Blocks

The following listing block contains `//` lines that are *code*, not comments,
and must be preserved exactly as written:

----
// This is a Ruby comment inside a listing — keep it!
def greet(name)
  puts "Hello, #{name}"
end
----

And a literal block:

....
// Not a comment — this is literal preformatted text.
  indented content
....
